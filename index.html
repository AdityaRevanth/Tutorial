How to Get a Domain and Set Its DNS Using Namecheap

Step 1: Go to Namecheap.com

Open your web browser and go to Namecheap's website: namecheap.com.

Step 2: Search for Your Desired Domain

Enter your desired domain name into the search bar on Namecheap's homepage and click "Search" to see if it's available.

Step 3: Add Domain to Cart and Checkout

If your domain is available, add it to your cart and proceed to checkout. Follow the steps to complete the purchase.

Step 4: Access Namecheap's DNS Settings

Log in to your Namecheap account, go to the domain management section, and find the DNS settings or 'Manage DNS' option.

Step 5: Update DNS Records

Here, you can add, edit, or delete DNS records like A, CNAME, MX, etc., to configure your domain settings.

Step 6: Configure GitHub Repository for Custom Domain

Open your GitHub repository settings for your GitHub Pages site.

Step 7: Add Custom Domain in GitHub Repository Settings

Scroll down to the "Custom domain" section in repository settings and enter your custom domain (e.g., example.com).

Step 8: Save Changes

Save the changes in your GitHub repository settings to apply the custom domain.

Step 9: Verify Domain Configuration

Follow GitHub's instructions to verify the domain configuration (e.g., adding a file or record to your repository).

Step 10: Wait for DNS Propagation

DNS changes may take time to propagate across the internet. Be patient; it can take several hours to a day.

Step 11: Check Custom Domain

After DNS propagation, visit your custom domain (e.g., example.com) in a browser to view your GitHub Pages site.

Hope this tutorial helped! Always double-check settings before saving changes.
