How to Get a Free Domain with Namecheap using GitHub Student Developer Pack

Step 1: Access the benefits from the Student Developer Pack

GitHub Education, verify your student status, and apply for the Student Developer Pack.

Step 2: Scroll down until u find Namecheap offer and click on the highlighted

Once approved, access the Namecheap offer in the GitHub Student Developer Pack. Follow the instructions to claim your free domain.

Step 3: Choose and Register Free Domain

Browse available domains, select one, and add it to cart and complete the order to gain access to your Free Domain valid for 1 Year. Make sure to select the Github Pages option when checking out, and for the Mail, use the primary E-mail not the college E-mail.

Step 4: Access Namecheap's account Settings

After registering into Namecheap , Log in to your Namecheap account and click on the Set Up Your Github Account. Now go to Namecheap.com, CLick on 'account' on top right.

Step 5: Access Custom Domain's Settings.

Go to Domain Listings in Account, and click on Manage on ur New Custom Domain, and then click on Advanced DNS, Scroll down and click Add new record. On the new empty record, set the Record as 'CNAME Record' and set the host as 'www' and set the Target in the following format 'githubusername.github.io', and click on the Tick and save Changes.

Step 6: Link Domain to GitHub Pages

There will be a newly added Repository in ur github by namecheap. Remove custom domain from the settings in that created repo, delete the repo and add your new custom domain to any repositery u want. Note : You can only add custom domain to one repository at a time.

Now you have the custom domain created successfully..
