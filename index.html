How to Get a Free Domain with Namecheap using GitHub Student Developer Pack

Step 1: Sign up for GitHub Student Developer Pack

Visit GitHub Education, verify your student status, and apply for the Student Developer Pack.

Step 2: Access Namecheap Offer

Once approved, access the Namecheap offer in the GitHub Student Developer Pack. Follow the instructions to claim your free domain.

Step 3: Choose and Register Free Domain

Browse available domains, select one, and proceed to register it for free using the provided offer.

Step 4: Access Namecheap's DNS Settings

Log in to your Namecheap account, go to the domain management section, and access the DNS settings or 'Manage DNS' option.

Step 5: Configure DNS Records

Add necessary DNS records like A, CNAME, etc., if required, to set up your domain.

Step 6: Link Domain to GitHub Pages

In your GitHub repository settings for your GitHub Pages site, add your newly acquired free domain.

Step 7: Verify Domain Configuration

Follow GitHub's instructions to verify the domain configuration (e.g., adding a file or record to your repository).

Step 8: Wait for DNS Propagation

DNS changes may take time to propagate across the internet. Allow several hours to a day for this process.

Step 9: Check Custom Domain

After DNS propagation, visit your custom domain in a browser to view your GitHub Pages site.

Hope this tutorial helped! Always double-check settings before saving changes.
